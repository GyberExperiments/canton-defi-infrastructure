## 🎨 **ШАГ 2: Настройка брендинга Fin**

### 2.1 Переход к настройкам Fin
1. В левом меню найдите **"Settings"** (Настройки)
2. Выберите **"Fin"** из списка
3. Нажмите **"Configuration"** (Конфигурация)

### 2.2 Основная информация
**Скопируйте и вставьте следующие данные:**

**Имя ассистента:**
```

```

**Описание:**
```
AI-ассистент для покупки Canton Coin (CC) и поддержки клиентов 1OTC
```

### 2.3 Загрузка логотипа
1. Нажмите **"Upload logo"** (Загрузить логотип)
2. Выберите файл `fin-assistant-logo.svg` из папки проекта
3. Подтвердите загрузку

```

- ✅ `conversation.admin.replied`
- ✅ `conversation.admin.assigned`

**Секрет webhook:**
```
intercom_webhook_secret_minimal_stage_2025

## 🏷️ **ШАГ 7: Настройка кастомных атрибутов**

### 7.1 Переход к атрибутам
1. В настройках Fin найдите **"Custom attributes"** (Кастомные атрибуты)
2. Нажмите **"Add attribute"** (Добавить атрибут)

### 7.2 Добавление атрибутов
**Добавьте следующие атрибуты:**

1. **order_id** (Текст)
   - Описание: ID заказа клиента

2. **order_type** (Текст)
   - Описание: Тип заказа (buy/sell)

3. **order_status** (Текст)
   - Описание: Статус заказа

4. **order_amount** (Число)
   - Описание: Сумма заказа

5. **canton_address** (Текст)
   - Описание: Canton адрес клиента

6. **preferred_network** (Текст)
   - Описание: Предпочитаемая сеть
