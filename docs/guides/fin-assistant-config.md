### 6. **Настройки интеграции**

#### **Webhook URL:**
```
https://stage.minimal.build.infra.1otc.cc/api/intercom/webhook
```

#### **События для отслеживания:**
```
• conversation.user.created
• conversation.user.replied
• conversation.admin.replied
• conversation.admin.assigned
```

### 7. **Кастомные атрибуты**

#### **Поля для отслеживания:**
```
• order_id - ID заказа клиента
• order_type - Тип заказа (buy/sell)
• order_status - Статус заказа
• order_amount - Сумма заказа
• canton_address - Canton адрес клиента
• preferred_network - Предпочитаемая сеть
• customer_tier - Уровень клиента
```
