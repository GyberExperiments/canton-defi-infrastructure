# 🎯 Простой способ получить сессию Telegram

**Цель**: Получить файл сессии для Telegram Client API

---

## ✅ САМЫЙ ПРОСТОЙ СПОСОБ (РЕКОМЕНДУЕТСЯ)

### Вариант 1: Python скрипт (Telethon) - САМЫЙ НАДЕЖНЫЙ

**Почему этот способ:**
- ✅ Работает стабильно
- ✅ Не требует интерактивного ввода (все через Telegram)
- ✅ Сессия сразу в правильном формате

**Шаги:**

1. **Установите Python и Telethon:**
   ```bash
   pip install telethon python-dotenv
   ```

2. **Убедитесь, что в `.env.local` есть:**
   ```bash
   TELEGRAM_API_ID=ваш_api_id
   TELEGRAM_API_HASH=ваш_api_hash
   ```

3. **Запустите скрипт:**
   ```bash
   python3 scripts/setup-telegram-session-telethon.py
   ```

4. **Следуйте инструкциям:**
   - Введите номер телефона (с кодом страны, например `+79991234567`)
   - Введите код из Telegram (придет в приложение)
   - Введите пароль 2FA (если установлен)

5. **Готово!** Сессия сохранится в `.telegram-session` и `.env.local`

### Вариант 2: Онлайн инструмент (БЫСТРО)

**Если Python не установлен или не работает:**

1. Откройте: https://sess.gogram.fun/
2. Введите `api_id` и `api_hash` (из my.telegram.org/apps)
3. Введите номер телефона
4. Введите код из Telegram
5. Скопируйте полученную сессию
6. Добавьте в `.env.local`:
   ```bash
   TELEGRAM_SESSION_STRING=скопированная_сессия
   ```

⚠️ **Внимание**: Используйте только проверенные инструменты!

## 🔍 ЧТО ДЕЛАТЬ ЕСЛИ НЕ РАБОТАЕТ?

### Проблема: "Telethon не установлен"

**Решение:**
```bash
pip install telethon python-dotenv
```

Или если pip не работает:
```bash
pip3 install telethon python-dotenv
```

### Проблема: "TELEGRAM_API_ID не найден"

**Решение:**
1. Откройте `.env.local` в корне проекта
2. Добавьте:
   ```bash
   TELEGRAM_API_ID=ваш_api_id
   TELEGRAM_API_HASH=ваш_api_hash
   ```

### Проблема: "Ошибка авторизации"

**Возможные причины:**
1. Неправильный номер телефона (нужен с кодом страны: `+7...`)
2. Неправильный код (код приходит в приложение Telegram, не в SMS)
3. Неправильный пароль 2FA (если установлен)

**Решение:**
- Проверьте номер: должен начинаться с `+` и кодом страны
- Проверьте код: он приходит в приложение Telegram на вашем телефоне
- Если 2FA включен, введите пароль правильно
