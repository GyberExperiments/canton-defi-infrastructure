## Промпт для анализа и улучшения дизайна

```

КОНТЕКСТ ПРОЕКТА:
- Next.js 15 с App Router
- Tailwind CSS + CSS переменные (дизайн-система в globals.css)

ct для иконок
- Ultra Modern 2025 стиль с glassmorphism эффектами
- Адаптивный дизайн (mobile-first)

ДИЗАЙН-СИСТЕМА ПРОЕКТА:
Изучи файлы:
1. src/app/globals.css - CSS переменные, градиенты, glass эффекты
2. docs/DESIGN_SYSTEM.md - документация дизайн-системы
3. tailwind.config.ts - конфигурация Tailwind с кастомными цветами
4. src/components/ui/Button.tsx - стили кнопок
5. src/components/ui/Input.tsx - стили инпутов

АНАЛИЗИРУЙ ВСЕ КОМПОНЕНТЫ:

Основные страницы:
1. src/app/page.tsx - главная страница
2. src/app/dex/page.tsx - страница DEX
3. src/components/IntegratedLandingPage.tsx - лендинг
4. src/components/ExchangeFormCompact.tsx - форма обмена
5. src/components/dex/SwapInterface.tsx - интерфейс swap
6. src/components/dex/BridgeInterface.tsx - интерфейс bridge
7. src/components/dex/NearWalletButton.tsx - кнопка кошелька

КРИТЕРИИ АНАЛИЗА:

1. ВЫРАВНИВАНИЕ И СПЕЙСИНГ:
   - Проверь все элементы на правильное выравнивание (align-items, justify-content)
   - Проверь spacing между элементами (gap, margin, padding)
   - Проверь вертикальный и горизонтальный ритм
   - Проверь отступы между секциями (должны быть консистентными)
   - Проверь выравнивание текста (text-align)
   - Проверь выравнивание иконок относительно текста

2. ИКОНКИ:
   - Проверь размеры иконок (должны быть консистентными: w-4 h-4 для маленьких, w-5 h-5 для средних, w-6 h-6 для больших)
   - Проверь выравнивание иконок относительно текста (flex items-center)
   - Проверь spacing между иконкой и текстом (gap-2, gap-3, mr-2)
   - Проверь использование правильных иконок из lucide-react
   - Проверь цвет иконок (должен соответствовать контексту)
   - Проверь что все иконки на своих местах и логично размещены

3. ТИПОГРАФИКА:
   - Проверь иерархию заголовков (h1 > h2 > h3)
   - Проверь использование fluid typography (text-fluid-*)
   - Проверь line-height для читаемости
   - Проверь font-weight консистентность
   - Проверь использование gradient-text где нужно

4. ЦВЕТА И КОНТРАСТ:
   - Проверь использование CSS переменных (не хардкод!)
   - Проверь контраст текста на фоне (доступность)
   - Проверь использование semantic цветов (text-input-suffix, text-input-on-dark-suffix)
   - Проверь правильное использование цветов в разных контекстах (светлый/темный фон)

5. КОМПОНЕНТЫ UI:
   - Проверь консистентность кнопок (размеры, варианты, стили)
   - Проверь консистентность инпутов (стили, размеры, placeholder)
   - Проверь использование glass эффектов (glass-light, glass-medium, glass-ultra)
   - Проверь использование градиентов (должны соответствовать дизайн-системе)
   - Проверь border-radius консистентность

6. АНИМАЦИИ:
   - Проверь плавность анимаций (duration, easing)
   - Проверь использование shouldReduceAnimations для мобильных
   - Проверь микроинтеракции (hover, focus states)
   - Проверь анимации появления элементов

7. АДАПТИВНОСТЬ:
   - Проверь mobile-first подход
   - Проверь breakpoints (sm, md, lg, xl)
   - Проверь размеры элементов на разных экранах
   - Проверь адаптивную типографику

8. ВЕРСТКА:
   - Проверь использование Grid vs Flexbox (правильный выбор)
   - Проверь responsive grid (grid-cols-1 md:grid-cols-2)
   - Проверь использование max-width контейнеров
   - Проверь padding контейнеров на разных экранах

ЗАДАЧИ:

1. НАЙДИ ВСЕ ПРОБЛЕМЫ:
   - Выпиши все найденные проблемы с указанием файла и строки
   - Классифицируй по категориям (выравнивание, иконки, цвета, spacing и т.д.)
   - Определи приоритет исправления (критично, важно, можно улучшить)

2. СОЗДАЙ ПЛАН УЛУЧШЕНИЙ:
   - Предложи конкретные исправления для каждой проблемы
   - Укажи какие изменения в CSS переменных нужны (если нужны)
   - Предложи улучшения дизайна (новые эффекты, анимации, компоненты)

3. РЕАЛИЗУЙ ИСПРАВЛЕНИЯ:
   - Исправь все проблемы с выравниванием и spacing
   - Стандартизируй размеры иконок и их размещение
   - Улучши типографику и иерархию
   - Исправь проблемы с цветами и контрастом
   - Оптимизируй анимации
   - Улучши адаптивность

4. СОЗДАЙ ОБНОВЛЕННЫЙ ДИЗАЙН:
   - Примени принципы Ultra Modern 2025
   - Используй glassmorphism эффекты правильно
   - Добавь микроинтеракции где уместно
```
