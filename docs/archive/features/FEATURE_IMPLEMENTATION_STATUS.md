**Вариант 2: Использование сервисного бота (рекомендуется):**
```typescript
async createServiceChat(orderId: string, order: OTCOrder, operatorId: number): Promise<string> {
  const serviceBotUsername = process.env.TELEGRAM_SERVICE_BOT_USERNAME; // @TECH_HY_Customer_Service_bot
```
