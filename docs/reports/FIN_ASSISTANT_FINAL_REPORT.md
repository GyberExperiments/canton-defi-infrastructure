## 🔧 ТЕХНИЧЕСКАЯ АРХИТЕКТУРА

### **Поток обработки сообщений:**
```
Клиент → Intercom Fin → Webhook → AI Агент → Ответ клиенту
                    ↓
                Создание заказа → Google Sheets → Telegram уведомление
```

### **Компоненты системы:**
1. **Intercom Fin** - Интерфейс для клиентов
2. **Webhook** - `/api/intercom/webhook` - Обработка входящих сообщений
3. **AI Агент** - `/api/intercom/ai-agent` - Обработка логики
4. **Google Sheets** - Хранение заказов
5. **Telegram** - Уведомления операторов

### **Безопасность:**
- ✅ **HMAC верификация** webhook подписей
- ✅ **Rate limiting** 10 запросов/минуту
- ✅ **PII маскирование** в логах
- ✅ **Админская авторизация** для API
